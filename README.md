### 应用简介

- 超轻量应用，大小在2M以内
- 支持markdown上传、编辑、导出
- GitHub风格渲染
- 支持Tex公式
- 支持HTML语法
- 设置各种常用编辑按键，方便手机端操作
- 每次退出应用后自动保存内容，无需担心丢失
